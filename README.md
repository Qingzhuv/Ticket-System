# 🎫 WESTCRAN 企业级工单管理系统 (V0.9)

本项目是由 **[WESTCRAN 西鹤软件](https://www.westcran.tech)** 开发的标准化官方体验反馈与投诉处理解决方案。旨在为企业构建一道严谨、透明且具备法律存证能力的对外沟通桥梁。

## ✨ 核心特性

- **官方闭环逻辑**：支持从“用户发起-验证码防刷-后台受理-实时回复”的完整工单生命周期管理。
- **进度双向透明**：用户凭工单号与提交日期即可精准追踪处理进度，支持补充异议，拒绝“石沉大海”。
- **极致本地化**：所有静态资源（CSS/JS）完全本地化，不依赖外部 CDN，即便在内网环境也能稳定运行。
- **一键初始化**：内置 `init_db.php` 脚本，自动化完成建库、建表及管理员预设，实现真正意义上的“开箱即用”。

## 🛠️ 环境要求

- **服务器**：PHP 7.4+ (必须开启 `GD` 扩展以支持验证码，开启 `PDO_MySQL`)
- **数据库**：MySQL 5.7+ / 8.0+
- **部署环境**：支持 Apache / Nginx，推荐使用宝塔面板或小皮面板。

## 🚀 快速部署步骤

### 1. 源码上传

将本项目代码上传至服务器 Web 根目录。

### 2. 数据库配置

打开 `db_config.php` 文件，填入您的数据库连接凭据：

PHP

```
$host = 'localhost';
$user = 'root';        // MySQL用户名
$pass = 'root123';     // MySQL密码
$db_name = 'corp_system';
```

### 3. 一键初始化

在浏览器中访问：`http://您的域名/init_db.php`。 系统会自动创建数据库、表结构，并初始化默认管理员账号。**部署成功后，请务必删除该脚本。**

## 📂 访问说明

- **前台门户首页**：`http://您的域名/index.php`
- **后台管理入口**：`http://您的域名/admin_login.php`
- **默认管理员账号**：`admin`
- **默认管理员密码**：`admin123` (请务必在正式环境修改)
